# Taller2-Redes-Neuronales

Este repositorio contiene la solución completa a los **cuatro puntos del Taller 2 de Redes Neuronales**, organizados en **cuatro archivos diferentes**:

* `punto1.ipynb`
* `punto2.ipynb`
* `punto3.ipynb`
* `punto4.ipynb`

Cada archivo es un **notebook de Jupyter** que aborda uno de los puntos del taller de manera independiente, incluyendo la implementación práctica y las visualizaciones correspondientes.

---

## Requisitos

Para ejecutar los notebooks correctamente, es necesario tener instalado **Python 3.8 o superior** y las siguientes dependencias:

```bash
numpy
matplotlib
jupyter
```

Puedes instalarlas fácilmente ejecutando:

```bash
pip install -r requirements.txt
```

---

## Ejecución

1. Clona este repositorio:

   ```bash
   git clone https://github.com/tu-usuario/Taller2-Redes-Neuronales.git
   ```
2. Entra al directorio del proyecto:

   ```bash
   cd Taller2-Redes-Neuronales
   ```
3. Abre Jupyter Notebook:

   ```bash
   jupyter notebook
   ```
4. Abre cualquiera de los archivos `punto1.ipynb`, `punto2.ipynb`, `punto3.ipynb` o `punto4.ipynb` y ejecútalos celda por celda.
